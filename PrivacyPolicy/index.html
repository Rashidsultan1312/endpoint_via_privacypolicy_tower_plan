1. Introduction
This Privacy Policy explains how TRP Team (“we”, “us”, or “our”) handles information when you use the TowerPlan mobile application (“App”).

2. App Description
TowerPlan is a planning application that allows users to:

- create and manage building or construction plans
- calculate project budgets and cost estimates
- organize materials, expenses, and project details
- track progress of construction-related tasks

The App is designed for personal use and works directly on the user’s device without requiring account creation.
3. How the App Works
The App allows users to manually input planning data such as:

- project details
- cost estimates
- materials and budget information
- notes and planning records

All calculations and data processing are performed locally on the user’s device.

The App does not require an internet connection and does not connect to external servers for data storage or processing.
4. Data Collection
TowerPlan does not collect, store, or transmit personal data.

All user-entered data:

- remains on the user’s device
- is not accessible to the developer
- is not shared with any third parties

The App:

- does not require registration
- does not request personal information (such as name, email, or phone number)
- does not create user accounts
5. Local Data Storage
The App stores information locally on the user’s device, including:

- project plans
- cost calculations
- material lists
- notes and user preferences

This data:

- remains stored only on the device
- is protected by standard device security mechanisms
- is not transmitted externally
6. Tracking and Analytics
TowerPlan:

- does not track user activity
- does not include third-party SDKs that collect data
- does not use advertising frameworks
7. What the App Does Not Do

- does not transmit data over the internet
- does not include ads

- does not include social features
