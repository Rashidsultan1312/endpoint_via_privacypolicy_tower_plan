Privacy Policy for RoadStep

Last updated: April 13, 2026

This Privacy Policy describes how RoadStep handles user information.
1. App Description
RoadStep is a fitness tracking application that allows users to:

- track daily steps
- monitor distance and calories burned
- set daily goals
- complete daily challenges
- view progress in a gamified format

The app works entirely on the user’s device and does not require account creation.
2. How the App Works
The app uses built-in device capabilities to access motion and activity data (such as step count).

All data processing and calculations are performed locally on the user’s device.

The app does not operate its own backend servers and does not transmit data to external servers.
3. Data Collection
RoadStep does not collect, store, or process any personal data.

The app:

- does not require registration
- does not request personal information such as name or email
- does not create user accounts
4. Motion &Activity Data
The app may request access to motion and fitness activity data (such as steps) through system frameworks (e.g., HealthKit or Motion &Fitness).

This data:

- is used solely to provide app functionality
- is not shared with third parties
- is not stored on external servers
5. Local Storage
The app may store certain information locally on the user’s device, such as:

- step progress
- completed challenges
- streaks
- user preferences

This data:

- remains on the device
- is not shared externally
6. Tracking and Analytics
RoadStep:

- does not use analytics tools
- does not use advertising frameworks
- does not track user behavior
7. What the App Does Not Do
The app:

- does not require account creation
- does not send user data to servers
- does not include advertising
- does not sell or share user data
- does not include social features (such as chat or comments)
8. Children
RoadStep is not specifically directed to children but may be used by a general audience.

The app does not knowingly collect personal data from children.
9. Changes to This Privacy Policy
We may update this Privacy Policy from time to time.

The current version will always include the latest update date.

Continued use of the app after updates constitutes acceptance of the revised policy.
